## 7. 审计意见与报表口径

- 审计意见：无保留意见。
- 是否通过：通过。
- 关键审计事项：发放贷款和垫款、以摊余成本计量的金融投资、信贷承诺及财务担保的预期信用损失计量；结构化主体合并评估。
- 报表口径：合并报表。交通银行是大型银行集团，子公司和境外机构纳入并表；单看母公司不能完整反映金融租赁、基金、理财、信托、保险、境外证券、债转股等综合化经营。

## 8. 公司业务概况

- 主营业务：商业银行为主体，提供存贷款、产业链金融、现金管理、国际结算与贸易融资、投资银行、资产托管、财富管理、银行卡、私人银行、资金业务等服务。
- 行业地位：始建于 1908 年，2023 年入选全球系统重要性银行；按一级资本排名，2025 年报披露为全球银行第 9 位。
- 核心资产/核心产品：客户贷款、金融投资、客户存款、资产托管、财富管理、科技金融、绿色金融、普惠金融、贸易金融。
- 经营规模：2025 年末资产 15.55 万亿元，客户贷款 9.12 万亿元，客户存款 9.31 万亿元；境内 2800 余家网点、境外 24 家分（子）行及代表处，服务 307 万公司客户和 2.05 亿零售客户。
- 收入结构：2025 年营业收入 2650.71 亿元，其中利息净收入 1730.75 亿元，占 65.29%；手续费及佣金净收入 381.83 亿元，占 14.40%。
- 商业模式：以存款和同业/债券融资为负债来源，通过贷款、金融投资、金融市场和综合化牌照赚取净息差、手续费与投资收益。
- 收入驱动因素：贷款规模、存款成本、资产收益率、净息差、手续费业务恢复、金融市场业务。
- 利润和现金流驱动因素：净息差、信用减值损失、成本收入比、不良贷款处置和资本约束。
- 主要财报证据：2025 年报第 7、11、19、37、52、142-152 页。

## 9. 资产负债表四步扫描

### 9.1 资产规模与杠杆

- 总资产：15.55 万亿元。
- 总负债：14.27 万亿元。
- 资产负债率：约 91.77%。
- 判断：银行业天然高杠杆，不能套用非金融企业 70% 警戒线；应重点看资本充足率、杠杆率、流动性覆盖率和资产质量。2025 年资本充足率 15.96%、核心一级资本充足率 11.43%、杠杆率 7.28%，均满足监管要求。

### 9.2 生产资产

- 纳入的生产资产：固定资产 2447.13 亿元、在建工程 22.13 亿元、无形资产 48.20 亿元。
- 生产资产/总资产：约 1.62%。
- 轻资产或重资产判断：银行不是制造业，经营核心资产是贷款和金融投资，不是固定资产。
- 主要风险：核心风险不是固定资产减值，而是贷款、金融投资和表外信用承诺的预期信用损失。

### 9.3 应收与存货质量

- 应收账款及应收票据：不适用。
- 应收/总资产：不适用。
- 存货：不适用。
- 趋势与风险判断：用贷款质量替代判断。2025 年末客户贷款 9.12 万亿元，同比增长 6.64%；不良贷款率 1.28%，较上年末下降 0.03 个百分点；拨备覆盖率 208.38%，较上年末提高 6.44 个百分点。

### 9.4 货币资金与有息负债

- 货币资金：现金及存放中央银行款项 7018.47 亿元。
- 有息负债：银行负债大多带息，主要包括客户存款 9.31 万亿元、同业存放 1.16 万亿元、拆入资金 5349.94 亿元、已发行存款证 1.40 万亿元、应付债券 6920.85 亿元等。
- 现金债务覆盖：普通企业算法不适用；若仅对市场化批发融资粗算，现金及央行存款/批发融资约 15.35%。
- 存贷双高检查：不适用。银行应看流动性覆盖率和净稳定资金比例；2025 年四季度流动性覆盖率日均值 123.02%，净稳定资金比例季末值 110.38%。

## 10. 利润质量

- 营收趋势：2021-2025 年营业收入分别约 2693.90、2567.96、2575.95、2598.26、2650.71 亿元，2025 年同比增长 2.02%；2026Q1 同比增长 4.89%。
- 毛利率趋势：银行不适用毛利率；用净利息收益率替代。2021-2025 年净利息收益率从 1.56%降至 1.20%，2026Q1 回到 1.23%。
- 净利率趋势：2025 年归母净利润/营业收入约 36.07%，盈利韧性仍在。
- 费用率趋势：2025 年成本收入比 29.30%，较 2024 年下降 0.60 个百分点。
- 非经常性损益：2025 年归母净利润 956.22 亿元，扣非归母净利润 952.82 亿元，非经常性损益影响很小。
- 归母净利润趋势：2021-2025 年约 875.81、921.02、927.28、935.86、956.22 亿元，低速增长；2026Q1 归母净利润 261.62 亿元，同比增长 3.11%。
- 利润质量判断：利润稳定性强，非经常性损益依赖低；主要压力是净息差长期下行和信用减值损失仍高。

判断：交通银行 ROE 低于历史两位数水平，估值折价有其原因；但低 PB、较高股息率和资本补充使下行保护较强。银行股的核心不是“高成长”，而是资产质量、资本安全垫和分红可持续性。

## 14. 派生指标与计算公式

- 资产负债率 = 总负债 14.268106 万亿元 / 总资产 15.548388 万亿元 = 91.77%。银行业不适用普通企业 70% 阈值。
- 客户贷款/客户存款 = 9.123571 万亿元 / 9.307815 万亿元 = 98.02%。
- 信用减值损失/营业收入 = 545.47 亿元 / 2650.71 亿元 = 20.58%。
- 归母净利率 = 956.22 亿元 / 2650.71 亿元 = 36.07%。
- 市净率普通股每股净资产口径 = 6.60 元 / 2026Q1 每股净资产 13.18 元 = 0.50 倍。
- TTM 归母净利润 = 2025 归母净利润 956.22 亿元 - 2025Q1 归母净利润 253.72 亿元 + 2026Q1 归母净利润 261.62 亿元 = 964.12 亿元。
- TTM EPS = 964.12 亿元 / 883.64 亿股 = 1.09 元。
- 当前 PE = 6.60 元 / 1.09 元 = 6.05 倍，与集思录 6.072 倍接近。
- 2025 年普通股现金分红率 = 286.92 亿元 / 归母净利润 956.22 亿元 = 30.00% 左右；年报披露占归属于母公司普通股股东净利润 32.3%。
- 本次股息率估算 = 286.92 亿元 / 5832.01 亿元 = 4.92%，与集思录 TTM 股息率 4.879% 接近。

## 15. 估值与击球区

银行股估值应以 PB、ROE、资产质量、分红共同验证；为满足本模板的 PE 击球区要求，本次用成熟银行 PE 场景法，并用 PB 和股息率交叉检查。

| 情景 | PE | 对应股价 | 说明 |
|---|---:|---:|---|
| 悲观 | 5.5 倍 | 6.00 元 | 净息差低位、不良率上行时的保守估值 |
| 合理 | 6.5 倍 | 7.09 元 | 国有大行低增长、高分红、资产质量稳定情景 |
| 乐观 | 7.5 倍 | 8.18 元 | 息差企稳、资产质量改善、资本充足率提升情景 |

理想买入价 = 合理价值 7.09 元 x 70% = 4.96 元。

| 区间 | 价格 |
|---|---:|
| 强击球区 | 小于等于 4.96 元 |
| 保守观察区 | 4.96-6.00 元 |
| 轻仓区 | 6.00-7.09 元 |
| 持有区 | 7.09-8.18 元 |
| 高估/减仓区 | 高于 8.18 元 |

当前 6.60 元位于轻仓区。PB 低于 0.5 倍、股息率接近 5% 提供一定安全边际，但 ROE 下行和银行资产质量滞后暴露风险决定了不宜给高估值倍数。

## 16. 主要风险与缺失证据

- 净息差压力：2025 年净利息收益率 1.20%，较 2021 年 1.56%明显下降；2026Q1 仅回升至 1.23%。
- 资产质量滞后风险：2026Q1 不良贷款率升至 1.30%，拨备覆盖率降至 202.80%，虽仍较强但边际转弱。
- 信用减值模型主观性：审计关键事项显示 ECL 参数和宏观假设依赖管理层判断。
- 资本约束与摊薄：2025 年完成 1200 亿元定向增发，总股本增至 883.64 亿股，资本改善但 EPS 被摊薄。
- 房地产、地方债、零售信贷等重点领域仍需持续跟踪。
- 缺失证据：未做完整同业横向估值分位；当前行情为 2026-05-26 非当日快照；2021-2022 下载文件为 H 股公告版，部分检索质量较弱。

## 17. 后续跟踪指标

- 净利息收益率是否稳定在 1.20%上方，存款成本是否继续下降。
- 不良贷款率、关注类贷款、逾期 90 天以上贷款占不良贷款比例。
- 拨备覆盖率是否保持 200%附近，若跌破 180%需重估安全边际。
- 核心一级资本充足率是否维持 11%以上，后续资本工具发行是否继续摊薄普通股回报。
- 营业收入和归母净利润是否能维持低个位数正增长。
- 分红率是否稳定，股息率是否仍能覆盖银行股风险溢价。
- 股价是否跌入保守观察区或强击球区，或高于乐观价值进入减仓区。

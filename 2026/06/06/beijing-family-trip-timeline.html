6月13日-14日 望京出发 妹妹一家四口 轻松版

北京周末行程时间线

12号晚到，周六不早起。第一天看胡同和北京夜生活，第二天把央美毕业展作为核心，再顺路去798。整体按“少折返、少暴晒、随时可撤退”安排。

6月13日 周六 多云/阴，约30-32℃。适合午后开始慢逛，注意防晒和补水。
6月14日 周日 更热，有小雨风险，部分预报到33-35℃。把央美和798放这天更舒服。

核心原则

周六睡到自然醒，周日看艺术。下午热的时候多进店、多打车。

预约

央美毕业季建议提前预约购票；簋街晚饭务必提前订位。

随身

护照原件、防晒、折叠伞、充电宝、舒服的鞋。

不推荐

这版不排长城。12号晚到后，周六早起远郊风险太高。

胡同慢逛 + 簋街晚饭

从望京出发，路线短、北京味足，适合恢复体力。

6/13 周六
11:00-12:00

自然醒，望京早午饭

不要急着进城。先吃点清淡的，检查证件、伞和充电宝。

12:30-14:00

国子监街 / 五道营胡同 / 雍和宫外圈

这一带比南锣鼓巷安静，适合先找胡同感觉。若有兴趣，可进孔庙和国子监博物馆；只想轻松就走街、拍照、喝咖啡。

建议走法：雍和宫附近下车 -> 国子监街 -> 五道营胡同 -> 咖啡/小店休息。
14:30-16:30

南锣鼓巷周边胡同

南锣主街人多，把它当入口就好。重点走周边支巷，会更像真正的北京胡同。

推荐路线：南锣鼓巷地铁站 -> 南锣主街短逛 -> 雨儿胡同 -> 帽儿胡同 -> 菊儿胡同 -> 鼓楼/钟楼外拍。
16:30-17:30

烟袋斜街 / 什刹海，或直接休息

有体力就走到银锭桥看水边傍晚。若天气闷热或人太多，直接打车去簋街附近坐下休息。

17:45-19:30

簋街晚饭

周六晚饭高峰会排队，建议订17:30-18:00的桌。小龙虾、烤鱼、北京菜都可以，氛围比精致餐厅更有北京夜生活感。

20:00以后

回望京

第二天看央美，别把周六拖太晚。想喝东西可以回望京附近解决。

央美毕业展 + 798艺术区

给17岁女孩留足看展时间，14岁男孩也有文创、装置和拍照点。

6/14 周日
09:00-09:25

望京出发去央美

中央美术学院美术馆在花家地南街8号，从望京过去很近。周日可能热，尽量上午进馆。

09:30-12:30

央美美术馆：焕彩共生毕业季

这次正赶上本科毕业展第二阶段，重点院系包括中国画、书法、版画、雕塑、人文、设计、建筑。建议不要赶，按兴趣慢看。

看展顺序：设计/建筑 -> 雕塑/版画 -> 中国画/书法 -> 文创店或咖啡短休。
12:30-13:45

午饭和降温

在花家地、望京或丽都附近吃饭。下午还有798，午饭不要安排太远。

14:15-16:30

798艺术区

从央美到798顺路。不要把798当步行拉练，选1-2个展馆或画廊，再逛文创和咖啡店即可。

轻松走法：打车到798核心区 -> 看当天感兴趣的展 -> 文创店 -> 咖啡/甜品。
16:30-18:00

雨天或疲劳时提前撤

如果下雨、闷热或大家看展疲劳，直接回望京休息。不要为了“逛完”消耗体力。

18:00以后

望京 / 丽都晚饭

晚饭放回东北边，避免周日晚跨城。可以选烤鸭、涮羊肉、北京菜或他们想吃的中餐。

雨天备选

  • 周六下雨：缩短南锣主街，改成五道营咖啡店 + 簋街早晚饭。
  • 周日下雨：央美不变，798只保留室内展馆和咖啡，不走户外。
  • 如果临时很想看经典景点，故宫可替换798，但周日热且折返更累。

预约清单

  • 央美美术馆：提前在官方渠道预约/购票，带好有效证件。
  • 孔庙和国子监：可通过官方公众号购票，也可当天按现场规则处理。
  • 簋街：提前订17:30-18:00的桌，避开排队高峰。
  • 所有人：护照原件随身带，尤其是外籍或从日本来访的家人。
信息参考： 央美美术馆 央美毕业季展览 本科毕业展阶段说明 孔庙和国子监 天气以出行前一天官方预报为准。
